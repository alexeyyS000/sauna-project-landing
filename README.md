# Sauna Rental & Bathhouse Leisure Landing Page

A landing page for a company providing sauna rental and bathhouse leisure services.

## Technologies

* Python
* Wagtail CMS
* Nginx
* HTML
* CSS
* JavaScript

## Implemented Features

* Designed the landing page structure and user flow;
* Developed a fully responsive interface for mobile and desktop devices;
* Created informational sections describing services, benefits, and contact details;
* Implemented intuitive page navigation;
* Optimized website performance and loading speed;
* Built a conversion-focused landing page with an emphasis on user experience (UX);
* Developed a flexible admin panel that allows administrators to manage and customize any section of the website without code changes.
